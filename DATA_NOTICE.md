# Data and Source Text Notice

This project uses proposition text derived from Ludwig Wittgenstein's Tractatus Logico-Philosophicus, using public-domain source material made available through Project Gutenberg.

The repository includes derived English-only and German/English-aligned JSON datasets. These datasets contain parsed proposition text plus hierarchy metadata inferred from the Tractatus numbering system.

Project Gutenberg materials are generally public domain in the United States, but users outside the United States should check the copyright status of the underlying text and translations in their own jurisdiction.
