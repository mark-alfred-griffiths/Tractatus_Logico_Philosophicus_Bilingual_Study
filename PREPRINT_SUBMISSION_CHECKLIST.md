# Preprint Submission Checklist

- [x] PDF compiles without errors.
- [x] Title, author, affiliation, abstract, and keywords appear clearly in the PDF.
- [x] Manuscript states that it is a preprint.
- [x] No JDDH accepted-article template has been applied.
- [x] All figures and tables are present and cited.
- [x] All numerical results match the source manuscript.
- [x] References are complete and consistently formatted.
- [x] Project Gutenberg is cited as the text source.
- [x] Code availability section is present.
- [x] Licence choice is recorded for HAL.
- [x] No unsupported claims of philosophical understanding or general language-invariant logic remain.
- [x] HAL deposit should be registered as preprint / working paper, not as journal article.
