Monolingual Results Summary: English Split-Latent Tractatus Study

This file is a short retained summary of the English-only split-latent study. It
is supporting material for the canonical paper:

    paper/bilingual_structure_latents_in_the_tractatus.txt

The canonical monolingual run is the balanced English split-latent configuration
repeated across ten seeds:

    runs/seed_sweeps/monolingual_split_24_8_reg005/

Dataset:

    tractatus_structure_latents/data/tractatus.json

Model:

    z_text:        24 dimensions
    z_structure:   8 dimensions
    beta_text:     0.01
    beta_structure: 0.05
    seeds:         0..9

Purpose:

    The English-only study tests whether the Tractatus numbering system can be
    used as weak supervision for learning a compact latent representation of
    formal proposition position. The split-latent design separates text
    reconstruction from hierarchy prediction, making the 8-dimensional
    z_structure subspace interpretable as the structural channel.

Current ten-seed summary:

    parent accuracy:       0.6983 +/- 0.0152
    depth accuracy:        0.9274 +/- 0.0275
    next accuracy:         0.3724 +/- 0.0198
    reconstruction loss:   1.2923 +/- 0.0194
    perplexity:            3.6416 +/- 0.0706
    KL structure:          8.9653 +/- 0.1127
    sibling distance:      2.0656 +/- 0.0342
    parent-child distance: 6.1065 +/- 0.1566
    unrelated distance:    5.5574 +/- 0.0532

Interpretation:

    The split-latent model learns a structure subspace that carries hierarchy
    information from the Tractatus numbering system. Depth prediction is strong,
    parent prediction is useful, and next-proposition prediction remains the
    hardest structural target. The latent distance metrics support the same
    interpretation: sibling propositions are much closer to each other than
    unrelated propositions in z_structure.

What this result does not show:

    It does not show that the model understands Wittgenstein. It does not show
    unsupervised discovery of the hierarchy from raw text alone. The hierarchy is
    supplied as weak supervision through parent, depth, next, and child-count
    targets. The result is narrower: a compact 8-dimensional structure latent can
    represent formal proposition position when separated from the text
    reconstruction latent.
